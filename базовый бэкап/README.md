### Сохраненные файлы конфигурации

- `package.json` - зависимости проекта
- `package-lock.json` - зафиксированные версии зависимостей
- `tsconfig.json` - конфигурация TypeScript
- `next.config.js` - конфигурация Next.js
- `.env.example` - пример переменных окружения
- `.env.local` - локальные переменные окружения (если есть)
- `.gitignore` - правила игнорирования файлов
- `prisma/` - схема базы данных и миграции Prisma

## Как восстановить из бэкапа

1. Скопируйте файлы из этой папки в корень проекта
2. Установите зависимости: `npm install`
3. Если используется Prisma: `npx prisma generate`
4. Запустите проект: `npm run dev`

## Примечания

- Этот бэкап создан для быстрого восстановления рабочей конфигурации
- При восстановлении убедитесь, что версии Node.js и npm совместимы
- Проверьте актуальность переменных окружения в `.env.local`
