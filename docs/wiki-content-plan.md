# Контент-план для вики-инструкции логистов

## Структура вики

### 1. Главная страница (introduction)
- Культура и структура компании
- Цели, ценности и роль логиста
- Философия премиум-сервиса (AJTBD, Клиентократия, Product Heroes)

### 2. Ежедневное расписание (daily-schedule)
- Детальное расписание 8:00-20:00
- Примеры задач для каждого временного блока
- Приоритизация задач

### 3. Полезные ссылки (useful-links)
- Все ссылки на чаты, таблицы, системы
- Организация по категориям
- Быстрый доступ

### 4. Контакты (contacts)
- Полный список контактов с ролями
- Способы связи
- Внутренние ссылки на процессы

### 5. Аккаунты (accounts)
- Учетные данные и доступы
- Инструкции по входу
- Безопасность

### 6. Примеры сообщений и скрипты (scripts)
- База скриптов для разных ситуаций
- Примеры общения с клиентами
- Шаблоны сообщений

### 7. Ценность отдела логистики (logistics-value)
- Детальное описание ценности отдела
- Метрики и KPI
- Вклад в бизнес

### 8. Метрики качества работы (quality-metrics)
- Количественная оценка без опросов клиентов
- Метрики по Клиентократии и Product Heroes
- Система премирования

### 9. Метрики общения (communication-metrics)
- Метрики для логистов
- Метрики для кладовщика
- Метрики для экспедиторов/сборщиков

### 10. Ключевое обещание клиенту (client-promise)
- Формулировка обещания
- Отслеживание выполнения
- Метрики выполнения

### 11. Процессы работы (work-processes)
- Согласование отгрузок
- Работа с оплатами
- Оформление заказов
- Работа с рекламациями

### 12. Регламенты и стандарты (regulations)
- Время реакции
- Качество коммуникации
- Документальное оформление

### 13. Работа с клиентами (client-work)
- Принципы общения
- Скрипты
- Обработка рекламаций

## Принципы контента

1. **AJTBD (Always Just The Best Delivery)**: Каждая доставка должна быть идеальной
2. **Клиентократия**: Клиент в центре всех решений
3. **Product Heroes**: Фокус на создании героев продукта через отличный сервис

## Метрики без опросов

- Время реакции на запросы
- Процент доставок в срок
- Количество повторных обращений
- Процент доплат принятых без возражений
- Время от готовности до доставки
