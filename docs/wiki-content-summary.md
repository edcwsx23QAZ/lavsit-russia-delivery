## Как использовать

### Инициализация контента в базе данных:

1. **Через API endpoint:**
   ```
   POST /api/wiki/init
   Body: { "overwrite": false } // или true для перезаписи
   ```

2. **Или через браузер:**
   - Открыть страницу вики
   - Использовать кнопку инициализации (если есть в UI)

### Редактирование контента:

Все разделы редактируемы через интерфейс вики:
- Кнопка редактирования для каждого блока
- Возможность добавления/удаления секций
- Возможность изменения структуры
