# Быстрый старт: Синхронизация с Google Sheets

## ✅ Что уже сделано

1. ✅ Создан менеджер синхронизации (`lib/delivery-crm/sync-manager.ts`)
2. ✅ Интегрировано отслеживание всех изменений:
   - Создание заказов
   - Обновление любых полей
   - Удаление заказов
   - Перемещение заказов (drag-and-drop)
   - Группировка/разгруппировка заказов
3. ✅ Автоматическая синхронизация при каждом изменении
4. ✅ История всех изменений

## 📋 Что нужно сделать вам

### Шаг 1: Создать Google Apps Script

1. Откройте таблицу: https://docs.google.com/spreadsheets/d/1lP2s2eTYWBqdKVsymlfBxDRioP_AiGxlTWGhYjiqBXk/edit

2. Перейдите: **Extensions → Apps Script**

3. Откройте файл `docs/google-apps-script-code.gs` в этом проекте

4. Скопируйте весь код из файла и вставьте в редактор Apps Script

5. **ВАЖНО**: Проверьте строку 15 в коде - там должен быть правильный ID вашей таблицы:
   ```javascript
   const SPREADSHEET_ID = '1lP2s2eTYWBqdKVsymlfBxDRioP_AiGxlTWGhYjiqBXk';
   ```
   Если ID другой - замените его!

6. Сохраните (Ctrl+S или Cmd+S)

### Шаг 2: Развернуть как Web App

1. В редакторе Apps Script нажмите **Deploy → New deployment**

2. Нажмите на иконку ⚙️ рядом с "Select type" → выберите **Web app**

3. Заполните:
   - **Execute as**: **Me**
   - **Who has access**: **Anyone** ⚠️ (важно!)

4. Нажмите **Deploy**

5. При первом развертывании:
   - Нажмите **Authorize access**
   - Выберите ваш Google аккаунт
   - Нажмите **Advanced → Go to [Project Name] (unsafe)**
   - Нажмите **Allow**

6. **Скопируйте Web App URL** (выглядит как: `https://script.google.com/macros/s/.../exec`)

### Шаг 3: Настроить в приложении

1. Создайте файл `.env.local` в корне проекта (если его нет)

2. Добавьте строку:
   ```
   NEXT_PUBLIC_GOOGLE_APPS_SCRIPT_URL=https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec
   ```
   Замените `YOUR_SCRIPT_ID` на URL, который вы скопировали

3. Перезапустите dev-сервер:
   ```bash
   npm run dev
   ```

### Шаг 4: Проверить работу

1. Откройте: http://localhost:9000/delivery-crm

2. Внесите любое изменение (например, измените адрес заказа)

3. Подождите 3-5 секунд

4. Откройте таблицу Google Sheets - данные должны обновиться

5. Проверьте лист "История изменений" - там должны быть записи

## 🎯 Что синхронизируется

- ✅ Все поля заказов (дата, номер, товары, адрес, контакт, оплата, время, комментарий)
- ✅ Флаги (Написали, Подтвердили, Отгрузили, Доставлен)
- ✅ Теги (REKL, ТК)
- ✅ Группировка заказов (groupId, позиция, размер группы)
- ✅ История всех изменений с timestamp

## 📊 Структура таблицы

### Лист "Доставки"
- Строка 1: Заголовки
- Строки 2+: Все заказы (полная синхронизация)

### Лист "История изменений" (создается автоматически)
- Строка 1: Заголовки
- Строки 2+: Все изменения с timestamp

## ⚠️ Важно

- Синхронизация происходит автоматически через 3 секунды после последнего изменения
- История изменений записывается сразу
- При удалении заказа в приложении он удаляется из таблицы
- Таблица полностью синхронизирована с приложением
