# Обновление Google Apps Script

## ⚠️ Важно: Обновите код Apps Script

Код Apps Script был улучшен для гарантированного отображения заголовков в Google Sheets.

### Что изменилось:

1. ✅ **Улучшена логика создания заголовков** - теперь заголовки всегда проверяются и создаются/обновляются при необходимости
2. ✅ **Добавлена обработка ошибок** для пустых листов
3. ✅ **Заголовки создаются для обоих листов**: "Доставки" и "История изменений"

### Как обновить:

1. Откройте вашу таблицу Google Sheets
2. Перейдите: **Extensions → Apps Script**
3. Откройте файл `docs/google-apps-script-code.gs` в этом проекте
4. **Замените весь код** в редакторе Apps Script на новый код из файла
5. Сохраните (Ctrl+S или Cmd+S)
6. **ВАЖНО**: Если вы уже развернули Web App, нужно обновить развертывание:
   - Нажмите **Deploy → Manage deployments**
   - Нажмите на иконку редактирования (✏️) рядом с вашим развертыванием
   - Нажмите **Deploy** (версия обновится автоматически)

### Проверка работы:

1. Откройте приложение: http://localhost:9000/delivery-crm
2. Внесите любое изменение (например, измените адрес заказа)
3. Подождите 3-5 секунд
4. Откройте таблицу Google Sheets
5. **Проверьте лист "Доставки"** - должна быть строка с заголовками (ID, Дата, № Заказа, и т.д.)
6. **Проверьте лист "История изменений"** - должна быть строка с заголовками (Время, Тип изменения, и т.д.)

### Примечание:

Ошибка "Script function not found: doGet" при открытии URL в браузере - это **нормально**. Браузер делает GET запрос, а скрипт обрабатывает только POST запросы. Это не влияет на работу синхронизации.
