# План заполнения контента вики по блокам

## Блок 1: Главная страница (introduction)
- Культура и структура компании
- Цели, ценности и роль логиста
- Философия премиум-сервиса (AJTBD, Клиентократия, Product Heroes)

## Блок 2: Ежедневное расписание (daily-schedule)
- Детальное расписание 8:00-20:00 с примерами
- Приоритизация задач
- Ссылки на контакты и процессы

## Блок 3: Полезные ссылки (useful-links)
- Ссылки на чаты Telegram
- Ссылки на Google Sheets
- Ссылки на системы (Битрикс24)
- Организация по категориям

## Блок 4: Контакты (contacts)
- Контакты команды логистики
- Контакты экспедиторов
- Контакты склада
- Контакты поставщиков
- Контакты транспортных компаний

## Блок 5: Аккаунты (accounts)
- Учетные данные Битрикс24
- Доступы к таблицам
- Инструкции по входу

## Блок 6: Примеры сообщений и скрипты (scripts)
- Скрипты для уведомления о готовности
- Скрипты для согласования доставки
- Скрипты для работы с доплатами
- Скрипты для работы с хранением

## Блок 7: Ценность отдела логистики (logistics-value)
- Описание ценности отдела
- Метрики вклада в бизнес
- Примеры создания ценности

## Блок 8: Метрики качества работы (quality-metrics)
- Количественные метрики без опросов
- Метрики по Клиентократии
- Метрики по Product Heroes
- Система премирования

## Блок 9: Метрики общения (communication-metrics)
- Метрики для логистов
- Метрики для кладовщика
- Метрики для экспедиторов/сборщиков

## Блок 10: Ключевое обещание клиенту (client-promise)
- Формулировка обещания
- Отслеживание выполнения
- Метрики выполнения

## Блок 11: Процессы работы (work-processes)
- Согласование отгрузок
- Работа с оплатами
- Оформление заказов
- Работа с рекламациями

## Блок 12: Регламенты и стандарты (regulations)
- Время реакции
- Качество коммуникации
- Документальное оформление

## Блок 13: Работа с клиентами (client-work)
- Принципы общения
- Скрипты общения
- Обработка рекламаций
